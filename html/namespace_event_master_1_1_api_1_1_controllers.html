 Controller for managing event reviews and organizer replies. Customers can create reviews for events they attended, and organizers can reply to reviews for their events. All users can view reviews and replies. The controller includes endpoints for creating reviews, checking review eligibility, retrieving reviews by ID or by event, and creating organizer replies.
