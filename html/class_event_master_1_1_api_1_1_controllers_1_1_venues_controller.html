Detailed Description

Controller for managing venues. This includes endpoints to retrieve a list of all venues and to get details of a specific venue by its ID. Both endpoints are public and do not require authentication, allowing anyone to view venue information. The controller uses the EventMasterDbContext to access the venues data from the database and returns the relevant information in the response. This allows users to browse available venues when planning events.
